Chapter No 1

Self-check quiz

  1. How do you link an external CSS file to an HTML document?

    Answer 1: We can link an external CSS file to an HTML document using

    <link rel="stylesheet" href="styles.css">

  2. What is the purpose of the 'style' attribute in HTML tags?

    Answer 2: The 'style' attribute is used to add inline CSS styles directly to an HTML element.

    <p style="color: blue; font-size: 16px;">This is a blue text with a font size of 16px.

  3. How do you write a CSS comment?

    Answer 3: CSS comments are written using '/*' to start the comment and to end it use '*/'.

    /* This is a CSS comment */

  4. What is the 'box-sizing' property in CSS used for?

    Answer 4: The 'box-sizing' property in CSS is used to control how an element's total width and height are calculated. The 'box-sizing' property can be set to 'content-box' (default) or 'border-box'. When set to 'border-box', padding and borders are included in the element's dimensions.
